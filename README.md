# KisanSaathi - Modern Agricultural Intelligence Platform

A comprehensive, enterprise-grade agricultural platform built with Next.js, featuring AI-powered crop advisory, pest detection, and market intelligence for Indian farmers.

## 🌟 Features

### Core Platform
- **Modern Next.js Architecture**: Server-side rendering and optimal performance
- **Responsive Design**: Mobile-first approach with seamless cross-device experience
- **Bilingual Support**: Full English/Hindi language support with easy toggle
- **Premium UI/UX**: Professional design with smooth animations and transitions
- **Complete Authentication**: Secure login and registration system
- **Comprehensive Dashboard**: Real-time agricultural intelligence and analytics

### Agricultural Intelligence
- **AI Crop Advisory**: Machine learning-powered recommendations
- **Smart Pest Detection**: Computer vision for disease identification
- **Weather Analytics**: 15-day forecasts with risk assessment
- **Market Intelligence**: Real-time pricing and demand forecasting
- **Voice Assistant**: Multilingual voice interface for hands-free operation
- **Government Integration**: e-NAM, PMFBY, and state scheme connectivity

## 🎨 Design System

- **Primary Color**: Deep Green (#0A6B3A) - Agriculture and growth
- **Accent Colors**: Gold/Yellow gradients for highlights and warmth
- **Typography**: Inter font family for modern, clean typography
- **Design Language**: Professional, trustworthy, and farmer-friendly
- **Animations**: Framer Motion for smooth, engaging interactions

## 🚀 Quick Start

### Prerequisites
- Node.js (version 16 or higher)
- npm or yarn package manager

### Development Setup

1. **Install Dependencies**:
```bash
npm install
```

2. **Run Development Server**:
```bash
npm run dev
```

3. **Access the Application**:
Open your browser and navigate to `http://localhost:3000`

### Production Build

```bash
# Build for production
npm run build

# Start production server
npm start
```
